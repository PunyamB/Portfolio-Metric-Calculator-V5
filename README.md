Real-time portfolio analytics dashboard for a $1M paper trading account. 
Computes risk metrics (Sharpe, Sortino, VaR, Beta, Alpha), Fama-French 5-factor regression, Markowitz mean-variance optimization, and generates AI-powered DOCX reports via Gemini API. 
Built with Python, Streamlit, and Yahoo Finance.
